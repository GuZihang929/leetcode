# 题目大意
    两个字符串，s，t。s[i]==t[i]时有匹配度，
    现在可以最多一次对t 中下标i与j的字母进行交换。
    求s和t最大的字符匹配度

## 例子

### 输入描述：

```
第一行输入一个整数
第二行输入一个长度为的字符串。
第三行输入一个长度为的字符串。
```

### 输出描述：

```
输出一个整数，和的最大匹配度。
```

### 示例1

输入例子：

```
5
ababc
babac
```

输出例子：

```
3
```

## 解题思路

    遍历字符串，字符相同，res++。
    不相同，用map保存，两个字符串不同的，key为byte，value为下标。
